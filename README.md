# Multithreaded Python Port Scanner

A fast and efficient multithreaded port scanner built using Python for network reconnaissance and security testing. This tool scans open TCP ports on a target system using socket programming.

---

## Features

- Fast multithreaded scanning
- TCP port scanning
- Supports IP address and hostname
- Lightweight and easy to use
- Built using Python socket and threading modules

---

## Technologies Used

- Python 3
- Socket Programming
- Multithreading
- Networking Concepts
